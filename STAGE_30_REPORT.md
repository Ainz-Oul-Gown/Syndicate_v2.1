# Stage 30 — Повторная отправка неотправленных сообщений

## Реализовано

- В текущем чате отображается компактная панель, если есть неотправленные сообщения и соединение восстановлено.
- Кнопка «ОТПРАВИТЬ ВСЕ» повторяет отправку только сообщений текущего чата.
- Повтор выполняется последовательно, чтобы не перегружать мобильное соединение и снизить риск дубликатов.
- Во время массового повтора кнопка показывает состояние загрузки, а одиночные кнопки повтора временно блокируются.
- Текстовые и голосовые сообщения поддерживаются; для голосовых сохраняется исходный reply-контекст.
- После выполнения показывается итоговое уведомление: полный успех, частичный успех или ошибка.
- При повторной потере сети процесс прекращается, оставшиеся сообщения сохраняют статус «не отправлено».
- Существующая одиночная кнопка «Повторить» сохранена.

## Совместимость

Решение не требует серверных изменений и работает в браузере, GitHub Pages, standalone PWA и Android WebView/PWA-to-APK.
